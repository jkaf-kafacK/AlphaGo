# Dossier de l'implémentation de AlphaZero pour le jeu de Go

Le dossier **bibli/go_starter_pack** contient le module **MCTS** avec les fichiers 
de l'implémentation du MCST, pour run une partie la commande python est : 
python3 namedGame.py myPlayer.py 

Les fichers du dossier notebooks contiennent le début de l'implémentation du réseaux de 
neurones pour le AlphaZero. **data_parsing.ipynb** contient des fonctions utils pour le parsing des données et **pv_network.ipynb** contient le réseau de neurones pour la value et policy iteracy.
